Sobre o projeto

A EduTeca é um projeto com objetivo de desenvolver uma plataforma em página web, para divulgação de livros das bibliotecas do município de Indaiatuba. Entretanto, o propósito não é criar um catálogo que ofereça o conteúdo desses materiais de forma digital, propriamente dito; mas especificamente apenas os principais dados sobre, ou seja: título, autor, editora... Ademais, o site apresentará a(s) localidade(s) em que essa obra literária pode ser encontrada, logicamente, desde que esteja disponível entre os estabelecimentos participantes.
